# DevNotes MVP Spec

## 1. Product Summary
DevNotes là ứng dụng web ghi chú dành cho developer, hỗ trợ lưu trữ kiến thức kỹ thuật, code snippets, bug fixes và learning notes.

## 2. Target Users
- Developer cá nhân
- QA / Automation Tester
- Người học lập trình

## 3. Problems
- Ghi chú nằm rải rác nhiều nơi
- Khó tìm lại snippet hoặc bug fix cũ
- Công cụ note hiện tại chưa tối ưu cho markdown và code

## 4. MVP Goals
- Người dùng có thể đăng ký và đăng nhập
- Người dùng có thể tạo, sửa, xóa note
- Người dùng có thể gắn tag cho note
- Người dùng có thể tìm kiếm note theo tiêu đề và nội dung
- Người dùng có thể ghim note quan trọng

## 5. Out of Scope
- Realtime collaboration
- AI features
- Rich block editor
- Team workspace
- File attachments
- Offline sync

## 6. Main Entities
- User
- Note
- Tag

## 7. Main User Flows
- Đăng ký tài khoản
- Đăng nhập
- Tạo note mới
- Chỉnh sửa note
- Gắn tag
- Tìm kiếm note
- Ghim note

## 8. Success Criteria
- Có thể dùng app để lưu note hằng ngày
- CRUD note hoạt động ổn định
- Tìm kiếm ra note nhanh
- UI đủ sạch để dùng thật
